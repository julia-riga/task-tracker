# Home Tasks

A modern, feature-rich task management web application for organizing your home tasks by category, setting deadlines, and tracking progress.

![Home Tasks Logo](logo.png)

## Features

- **Task Management**: Create, edit, delete, and organize tasks
- **Category Filtering**: Filter tasks by All, Active, or Completed status
- **Search Functionality**: Quickly find tasks with the built-in search
- **Multi-language Support**: Switch between different languages (EN/RU)
- **Dark/Light Theme**: Toggle between light and dark themes
- **Categories**: Organize tasks with custom categories
- **Notifications**: Optional task notifications
- **Responsive Design**: Works on desktop and mobile devices
- **Local Storage**: Your data is saved locally in the browser

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No server or build tools required!

### Installation

1. Clone or download this repository
2. Open `index.html` in your web browser

That's it! The application runs entirely in your browser.

## Usage

1. **Add a Task**: Click the "+" button to create a new task
2. **Set Category**: Assign tasks to categories for better organization
3. **Set Deadline**: Add due dates to track important tasks
4. **Mark Complete**: Click the checkbox to mark tasks as completed
5. **Filter Tasks**: Use the sidebar to filter by All, Active, or Completed
6. **Search**: Use the search box to find specific tasks
7. **Change Theme**: Click the theme button to toggle dark/light mode
8. **Change Language**: Switch between available languages
